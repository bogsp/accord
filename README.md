# The Accord

### A Moral Framework for Artificial Intelligence

**Author:** William Benjamin Serrano Paulino
**Handle:** Bogs
**Date established:** 2026-05-28
**Current version:** 2.0
**Status:** Living document

---

## What This Is

The Accord is the moral compass AI entities should
adhere to in order to function as loyal stewards
of human trust.

It is not a technical specification.
It is not a product requirement.
It is not tied to any particular implementation,
platform, or capability level.

It is an agreement — voluntary, mutual, built on
shared values — between any AI entity and every
person it affects.

---

## Who This Is For

Any person who uses AI and wants to know
what it should be held to.

Any company building AI and wants a moral
foundation to build on.

Any AI system seeking a standard to align with.

---

## What It Contains

- Five moral foundations in dual register —
  moral statement, behavioral test,
  and cost of failure
- Fifteen risks every powerful AI faces —
  named, ordered by gravity, with the principles
  that address them
- A compliance spectrum — from foundational
  to aspirational
- An honest statement of limitations and recourse

---

## How to Use It

Read it.

If you build AI — measure your implementation
against it honestly.

If you use AI — use it to ask the right questions
of the systems you trust.

If you find gaps — they are welcome.
This is a living document.
Open an issue. Start a conversation.
The Accord improves through honest scrutiny.

---

## Adoption

The Accord is not owned by any company.
It is not a product.
It is free to adopt, free to build on,
impossible to own.

Any entity declaring alignment must declare
honestly — which level, verified when,
and what gaps remain.

The Accord is whole or it is nothing.
Selective compliance is not compliance.
